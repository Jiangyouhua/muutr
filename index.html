Forward 前言
目图是一款效率APP。也是我近年想要实现的第一个目标。
生活中，我们不断的有目标需要去实现，有自己的、自己与朋友的、自己与团队的。我一直想做一款APP，去帮助我们更好、更快的实现目标。
例如，最近我有健身的目标，目图能帮助我：
1、 分类管理：我以前有学习的目标，现在添加一个健身的目标。
2、 制定目标，确定目标的周期性：制定健身目标，确定每周一、三、五晚上8:30前，完成在健身房锻炼身体一个半小时。
3、 Check目标的每一个周期：每天，打开APP，目图都会提醒你，这一次的健身任务完成了吗？它还会提供至截止时间的倒计时？
4、 记录进程中的点滴：健身老师说，起步要慢一些；
5、 记下进程中的收支：买了一双跑步鞋，支出588元；
6、 能提醒你：这几天比较忙别忘了，于是在目图上设置任务截止时间前2个小时提醒自己；
7、 能帮你倒计时：目图告诉你，一个小时的跑步时间到了。该去做瑜伽了；
8、 能与志同道合的人一起前进：把蜜蜜加到健身类中来，我们一起来健身。类内有两个以上的伙伴就该称为群；
9、 能与共同利益的人组成团队：把我和蜜蜜的健身老师加到健身群中来；
10、能实现分工合作：健身老师可不需要跑步和做瑜伽，只需要盯着我们就行，所以他没有认领跑步和做瑜伽的任务；
11、能知道群里伙伴的任务状态：蜜蜜没有完成昨天的健身房一个半小时的健身任务呀；
12、能与群里的伙伴进行交流：谁知道，蜜蜜昨天为什么没去健身房锻炼呀？
13、能与群里的伙伴进行私聊：蜜蜜，我想跟你说个密秘；
14、能隐藏你的身份：群内伙伴只知道你的呢称或加密的账户；
于是，2017年4月份开始，我决定做这目图。
整整一年，才完成上线，提交上线的时间是2018年5月份。
我使用目图进行了它自己的上线测试管理，感觉比较好。

目图就在你身边。无论你是为了学习、工作、生活、健康、兴趣还是为了你的团队，它一直都在你身边。它可以帮助你制定并实施你的学习计划、工作规划、生活安排、健康习惯、兴趣方向、团队合作等。

Main Function 主要功能
目图支持着你。目图提供了群(分类)、任务（目标）、任务卡（目标卡）、聊天室（类备注）、笔记与收支、搜索任务（目标）、倒计时闹钟、二维码扫描等主要功能。目图帮助你去更好、更有效地安排、管理、执行你的任务（目标），让你更好地成长。

目图具有一定的封闭性：


1、目图里笔记与聊天室（类备注）中的内容不可复制。

2、目图里笔记与聊天室（类备注）中的文字、图片、照片、音频、视频、涂鸦、地理位置保存在目图里，不能被第三方软件引用。

3、目图只需要个人账号与密码信息，没有设置更多个人信息的功能，同时个人账号与密码信息不与其它用户共享。

4、目图里群内伙伴间的交流仅基于聊天室，没有与"添加好友"相似的功能。关闭群后，你与群里伙伴的关系恢复到原始状态。

Team Work 团队协作
目图支持团队协作。一个分类下面存在多个用户，分类就变为群，用户成为群里的伙伴，伙伴之间可以实现团队协作。伙们可以为团队添加任务，可以认领或放弃认领任务，可以查看任务中各个周期伙伴的完成情况，可以在聊天室一起沟通交流，可以@部分伙伴进行私聊。这些功能简单的实现了群内伙伴间的团队协作。
1. 进入群设置页：首页，点击页面顶部工具栏中的+号按钮，进入群设置页。 2. 添加群：群设置页，点击标题栏下的+号按钮，可新添一个群。 3. 设置群：群设置页，点击群名称可修改群名称；点击排序可排序群名称，该顺序影响到首页标签栏中群名称的顺序；左滑群名称显示删除按钮，点击删除按钮，删除或关闭该群。 4. 进入任务页：首页，添加群后，可以看到标签栏上不同颜色的群名称标签。其下区域为任务区，可点击右下角的+号按钮，进入任务设置页。 5. 添加任务：任务页，在加号标签下可新建一个任务。新建任务时有任务说明、重复周期、开始日期、结束日期、截止时间等可设置。
6. 管理任务：任务页，在心号标签下显示当前群的所有任务。点击各栏右则认领按钮可以领取该任务；点击各栏右则己领按钮可以放弃领取该任务；左滑己领任务显示编辑按钮，点击编辑按钮重新编辑该任务；左滑认领任务显示关闭按钮，点击关闭按钮关闭该任务，该任务将对你不可见。 7. 任务卡：首页，添加任务后，标签栏下的任务区中显示有任务卡。任务卡左则为最近任务状态（倒计时/OUT、OK两种状态），右则为任务说明、任务的起止时间、重复周期的日期、最近周期的日期、最近周期的截止时间、提醒时间及伙伴完成名单。 8. 聊天室：首页，点击任务区右下角的信息框按钮,进入群聊天室页。群内伙伴可在此发送文本、图片、照相、录音、录频、涂鸦、地理位置为内容的信息，并为录音信息提供未听标识。 9. @伙伴：聊天室页，点击顶部标题栏进入@伙伴页。@伙伴页中心区为伙伴名称，你可以选择需要@的伙伴。从@伙伴页返回聊天室页时，标题栏显示部分已选伙伴名单，现在发送的信息，只对选择的伙伴可见，即实现私聊。 10.加入群：聊天室页，点击顶部右上角更多按钮进入群信息页。在群信息页中有两种方式让伙伴加入群：
1. 点击群二维码栏中的显示按钮显示群二维码，伙伴通过目图的二维码扫描功能可实现加入群；
2. 点击分享给朋友栏中的分享按钮分享群信息至第三方，通过安装目图加入群。
Tracking goals 跟踪任务
目图全力为你跟踪任务。设置好的任务，目图会随着时间的推进，按各个任务设置的重复周期项生成任务卡。任务卡具有周期性、时效性。其中，首页的任务卡需要你Check当前周期的任务状态；周期页的任务卡需要Check历史周期的任务状态。目图还为各任务提供提醒功能，在预定的时间提醒你，该去完成当前周期的任务。
1. 加载任务卡：首页，标签栏下方有时间轴。时间轴的时间跨度为6个月（半年），任务区加载当前任务周期在这6个月内的首页任务卡。左右滑动时间轴，可改变时间轴上的时间，同时对应改变了任务区上加载的首页任务卡。 2. Check当前任务：首页，任务卡左则显示最近任务状态（倒计时/OUT、OK两种状态），完成该周期任务显示OK，未完成则显示至本期截止时间的倒计时，超时的显示OUT。点击最近任务状态确定是否完成最近的周期任务。 3. Check历史任务：首页，点击任务卡右则进入笔记页。在笔记页里切换到周期标签，周期标签下显示的是周期任务卡。周期任务卡与首页任务卡相似，点击周期任务状态确定是否完成该周期任务。 4. 添加提醒：笔记页，在笔记页里切换到提醒标签，提醒标签下可添加、修改任务提醒，已设置任务提醒的，可点击提醒说明右则的删除按钮删除该任务提醒。提醒在各周期截止时间前24小时以内以系统信息的方式发送。 5. 关闭所有提醒：关于页，点击首页工具栏最左关于按钮进入关于页。在关于页点击后台提醒栏的开关按钮，可关闭或开启已设置的所有提醒。
Note And Money 笔记与收支
目图笔记与收支功能，支持以任务为基础添加笔记与收支。可以添加文字、录音、照相、录频、涂鸦、地理位置等信息作笔记。同时可在各笔记上添加收支，添加的收支自动生成月报表，可查看各报表。支持对笔记的删除，删除笔记同时删除了收支。
1. 进入笔记页：首页，点击任务区任务卡右则，进入笔记页。在笔记页切换到笔记标签，笔记标签下显示该任务的最近笔记，上拉可加较早前笔记。 2. 添加笔记：笔记页，笔记标签下为编辑区，编辑区提供以文字、录音、照相、录频、涂鸦、地理位置信息作笔记的功能。已添加的笔记按提交时间的近远从上到下排列。 3. 添加收支：笔记页，编辑区下方是笔记区，在笔记区上点击笔记卡右上角文字，可为该笔记添加或修改收支的金额。 4. 删除笔记：笔记页，编辑区下方是笔记区，在笔记区上点击笔记卡左上角删除按钮，可删除该笔记，删除笔记同时删除了该笔记上的收支。 5. 查看收支报表：笔记页，点击页面顶部左上角$号按钮进入收支报表页，收支报表页显示当前任务的收支月报表。也可在关于页查看收支总报表。
Other Function 其它功能
目图其它功能帮助你。提供任务搜索、24内倒计时闹钟及二维码扫描等功能.
1. 搜索任务：首页，点击工具栏的搜索按钮进入搜索任务页，在搜索任务页顶部的搜索框中输入任务说明中包含的关键词，可搜索相关任务。 2. 倒计时闹钟：首页，点击工具栏的闹钟按钮进入闹钟页，在闹钟页切换至+号标签,+号标签下可添加24小时间内的倒计时闹钟。 3. 查看倒计时闹钟：闹钟页，在闹钟页切换-号标签,-号标签下可查看所有有效的闹钟。列表左则有说明信息、倒计时时间、发起时间与结束时间。 4. 删除倒计时闹钟：闹钟页，在闹钟页切换-号标签,-号标签下列表右则有删除按钮与重启按钮，点击删除按钮可删除该闹钟，点击重启按钮可重启该闹钟。 5. 扫描二维码：首页，点击工具栏的扫描按钮进入二维码扫描页。二维码扫描页中心区域有扫描区，对准二维码可实现二维码扫描。
